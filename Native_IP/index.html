住宅IP和数据中心IP(广播和非广播)

本文最后更新于：2023年12月15日 下午

简介

住宅IP：是由VerizonFiOS、AT＆TU-verse、Cox、Comcast、Charter或TimeWarner等（本地运营商）公司拥有的IP，它们可用于真人家庭。IP地址是由互联网服务提供商（ISP）分配给用户。绑定到家庭地址的IP地址叫住宅IP地址。也叫原生、本地IP、或非广播IP、Native IP。

（国内移动、联通、电信就是一级本地运营商）

其他类型

当然，市场上还有其他类型的产品：

1、机房数据中心代理:使用数据中心(机房)拥有和管理IP代理，俗称机房IP代理。（如阿里云、华为云提供的IP）

2、4G移动代理：由移动运营商分配给移动设备的真实4G IP地址。（蜂窝网络）

3、隧道转发：基础IP我们可以将代理分为两类:轮换代理和非轮换(粘性固定)IP地址。

机房ip和住宅ip有什么区别？

住宅IP：

住宅IP是ISP(互联网服务提供商)或ADSL实际住宅由供应商分配给个人用户IP地址，100%WIFI由网络真实用户组成，独家居住IP是真正的用户设备IP，用于普通网民IP完全一致。

住宅IP与机房相比，它具有连号、散点分布、与普通网民分布一致的特点IP,成功率采集、补量业务、SEO需要大量短效动态P的业务优势很大。

机房IP：

机房IP是机房中心，机房有很多服务器IP具有IP在线固定稳定，价格相对便宜，但机房IP连号、区域分布不平衡等。适用于挂机、下载、伪装访问等需要长期在线的业务。

区别总结：

机房IP稳定性高，在线时间长，价格相对较低。

住宅IP目标服务器无法检测到，因此无法屏蔽您IP，处理后的数据不能发送给你。

为什么要住宅IP

因为某些流媒体、游戏、APP会对IP类型进行区分对待，如Netflix、Tiktok 直播等等。

如何判断一个ip是机房ip还是住宅ip呢

原生 IP 就是指该 IP 的注册地址与其BGP ASN所在位置一致的 IP。

可以用一个网站来判断（http://ipinfo.io），type=ISP 代表为住宅 ，type=hosting 或者 type=business 为商用机房ip。

建议type类型为 双ISP比较好。

然后根据AS号查询归属地是否和IP地址一样。查询网址：https://bgp.he.net/
